## This Portfolio

This repository is the source for my personal portfolio site — built with React 18, Vite 5, Tailwind CSS, and Framer Motion.

**Notable implementation details:**
- Pinned scroll experience timeline — 500vh sticky section, `useScroll` scrub, cinematic job transitions with `AnimatePresence`
- Shared layout project expand — Framer Motion `layoutId` morphs cards into full-screen modals
- Cursor ambient light — CSS radial gradient updated via `requestAnimationFrame` (zero React re-renders)
- Word-by-word text reveals — custom `SplitReveal` component with staggered clip-path animation
- Scroll velocity particles — Lenis scroll events boost particle repulsion radius in real time
- Global 3D perspective — `useMotionValue` + `useSpring` drives shared tilt across all skill cards

```bash
npm install
npm run dev      # localhost:5173
npm run build    # production build
```
